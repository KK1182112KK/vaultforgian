## Description

```text
VaultForgian is a desktop Obsidian plugin that brings Codex into study notes. I built it to turn lecture and paper context into reviewable note patches: Codex answers in context, proposes a diff with visible evidence, and waits for approval before writing to the vault. I used Codex to plan, implement, debug, and polish the plugin and learned how much trust depends on explicit review surfaces, not silent automation.
```
